# 👣 맛자국
Spring Boot Team Project - 맛집 추천 및 동행 찾기 사이트

## 📒프로젝트 개요
인플루언서 맛집, 사용자 검색 맛집 등의 정보를 제공하며, 동행 찾기 글을 통해 맛집 메이트를 구할 수 있는 기능을 구현

## 🗓️ 프로젝트 기간
2023-08-09 ~ 2023-09-15

- `java version 17`
- `Spring Boot 3.1.1`
- `MySQL`
- `Redis`
- `IntelliJ IDEA Ultimate`
- `Spring Security`
- `JWT token`
- `Spring Data JPA`
- `HTML`
- `Java Script`

- `Docker`
